# Project: Infinite Canvas "Star-Map" Flow Editor

**Role:** You are a Senior Frontend Architect.
**Goal:** Build a complete, working React application using Vite and Tailwind CSS.
**Aesthetic:** Sci-Fi / Space. Black background with moving white stars (antigravity feel).

## Tech Stack Constraints
1.  **Framework:** React (Vite)
2.  **Styling:** Tailwind CSS
3.  **Graph Library:** `@xyflow/react` (formerly React Flow).
4.  **Icons:** `lucide-react`
5.  **Layout Engine:** `dagre` (for auto-arranging nodes).

## Core Features & Requirements

### 1. The Canvas (Space Theme)
* Use the `<Background />` component from React Flow with `variant="dots"` and a black background color.
* Implement a "smooth" feel. The nodes should look like they are floating (white borders, black fill, glow effects).

### 2. Custom Node Component ("Dynamic Node")
* Create a custom node type.
* **Appearance:** Resizable black box with a thin white border.
* **Dynamic Text:** The text inside must scale/fit the box. If the user resizes the node to be huge, the font size increases. If small, it decreases. (Use a library like `react-textfit` or inline SVG scaling).
* **Handles:** minimal white dots for connections.

### 3. "Select Child" & Recursive Glow Logic (Critical)
* **Toggle:** Add a toggle in the top toolbar: "Select Path Mode".
* **Logic:** When this mode is ON and a user clicks a node (e.g., "Node A"):
    * Highlight "Node A".
    * Find all children connected to Node A.
    * Find all children of those children (recursively down the tree).
    * **Visual:** Apply a `drop-shadow(0 0 10px white)` (neon glow) to all selected nodes and the edges connecting them.

### 4. Right-Click Interaction (Context Menu)
* When a user right-clicks a node, show a custom menu:
    * **"Add Child (If)"**: Creates a new node connected to the current one with the label "If".
    * **"Add Child (Else)"**: Same, but labeled "Else".
    * **"Add Info"**: Adds a standard info node.
* **Automation:** The new node should automatically appear at a logical position (not overlapping).

### 5. Top Toolbar
* **Full Screen:** Toggle standard browser fullscreen.
* **Auto Arrange:** Use `dagre` to automatically organize the messy graph into a clean tree structure.
* **Import/Export:**
    * **Save:** Button to download the current graph (nodes + edges) as a `.json` file.
    * **Load:** Button to upload a `.json` file and restore the exact state.

## Code Output Requirements
